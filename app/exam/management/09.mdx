# Adaptace nových zaměstnanců

Promyšlený a řízený specifický program adaptačních a vzdělávacích aktivit pro nové zaměstnance.

- usnadňuje seznámení s:
    * pracovními úkoly
    * pracovními podmínkami
    * pracovním a sociálním prostředím
    * potřebnými znalostmi a dovednostmi

## Formy adaptace
* Formální
    - plánovaná
    - řízená útvarem MLZ, přímým nadřízeným
    - využití mentora, školitele, kouče
* Neformální 
    - spontánní
    - zapojení spolupracovníků
    - přirozená interakce s prostředím

## Roviny adaptace
- celá organizace
- organizační útvar/pracovní skupina
- vlastní pracovní místo

## Cíle adaptace
- seznámení s charakterem práce
- implementace do organizační kultury
- formování pracovních schopností
- dosažení požadované úrovně výkonu

## Nástroje adaptace
- orientační balíček
- písemné a elektronické materiály
- e-learning
- osobní doprovázení
- osobní rozhovory
- hodnotící schůzky
- help-line a metodická podpora
- technická podpora

## Průběh adaptace
1. Oficiální zahájení
    - sdělení plánu
    - stanovení pravidel
    - pověření zodpovědných osob
2. Jednotlivé adaptační aktivity
3. Systematické vyhodnocování
4. Zakončení procesu

### Délka trvání
- různá dle pozice:
    * několikaminutové zaškolení
    * 1-3 měsíce (běžné)
    * až 12 měsíců (komplexní pozice)

## Rizika adaptace
- podceňování adaptačního procesu
- pouze formální realizace
- absence systému adaptace
- příliš malý zájem o adaptovaného
- příliš velký zájem o adaptovaného

## Body na potítko

1. PODSTATA: řízený program aktivit, cíl = seznámení s prací + podmínkami + prostředím
2. FORMY: formální = plánovaná (MLZ + nadřízený + mentor), neformální = spontánní (spolupracovníci)
3. ROVINY: organizace → útvar/skupina → pracovní místo
4. NÁSTROJE: orientační balíček + materiály + e-learning + doprovázení + rozhovory + podpora
5. PRŮBĚH: zahájení → aktivity → vyhodnocování → zakončení, délka = 1-3 měsíce běžně
6. RIZIKA: podceňování + formalismus + absence systému + nevhodná míra zájmu o pracovníka
